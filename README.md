# 🛰️ Satellite Orbit Simulator Suite

A comprehensive Python-based satellite mission simulation suite developed to explore orbital mechanics, satellite mission planning, visualization, and mission control concepts through multiple progressive phases..

This project demonstrates the fundamentals of aerospace engineering, orbital dynamics, scientific computing, and data visualization using Python.

The project began as a simple orbital visualization and gradually evolved into a complete mission planning and mission control environment featuring:
---

## 🚀 Features

### 🌍 Orbit Designer
- Design custom satellite missions
- Input altitude, inclination, and eccentricity
- Calculate orbital parameters
- Generate mission reports

### 🛰️ 3D Orbit Designer
- Realistic 3D Earth visualization
- Inclined and elliptical orbits
- Animated satellite movement
- Orbit trail visualization
- Auto-rotating camera

### 🎯 Mission Control Pro
- Multi-satellite simulation
- Perigee and apogee visualization
- Collision proximity monitoring
- Real-time mission dashboard
- Event logging system

### 🤖 Mission Control AI
- Interactive satellite management
- Save and load missions
- Satellite health simulation
- Mission reporting
- Satellite database management

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/Rishikeshgudla/Satellite-Orbit-Simulator.git
```

Move into the project directory:

```bash
cd Satellite-Orbit-Simulator
```

Install dependencies:

```bash
pip install -r requirements.txt
```
