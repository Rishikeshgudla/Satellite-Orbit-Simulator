# 🛰️ Satellite Orbit Simulator

A Python-based satellite mission simulation suite featuring orbit design, 3D visualization, mission planning, and mission control tools.

## Features

- Orbit Designer
- 3D Orbit Visualization
- Multi-Satellite Mission Control
- Satellite Health Simulation
- Mission Reports
- Save/Load Missions
- Elliptical Orbit Modeling
- Collision Detection

## Technologies Used

- Python
- NumPy
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python src/orbit_designer.py
```
